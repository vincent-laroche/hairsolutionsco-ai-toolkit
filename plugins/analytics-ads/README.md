# Analytics & Ads Plugin

Data-driven analytics and advertising automation for Hair Solutions Co. — GA4 reporting, Google Tag Manager implementation, and Google Ads Scripts for intelligent campaign management.

## Skills Included

| Skill | Purpose |
|-------|---------|
| `google-analytics` | GA4 traffic analysis, e-commerce reporting, conversion tracking, Data API |
| `google-tagmanager` | GTM container management, tag implementation, data layer, tracking pixels |
| `google-ads-scripts` | Automated bidding, budget pacing, performance alerts, campaign reporting |

## What You Can Do

- Pull GA4 traffic and revenue reports via the Data API
- Set up GA4 events and mark conversions
- Implement and audit GTM tags, triggers, and variables
- Deploy Meta Pixel and GA4 via GTM on Shopify
- Automate Google Ads budget monitoring and bid adjustments
- Get emailed performance alerts when campaigns underperform
- Generate weekly ad performance reports automatically

## Required Environment Variables

```bash
GA4_PROPERTY_ID=properties/123456789
GA4_KEY_FILE=path/to/service-account-key.json
GOOGLE_ADS_ACCOUNT_ID=your-account-id
```

## Quick Start

**GA4 Report**: "Pull last 30 days sessions and revenue by channel"
**GTM**: "Set up a Meta Pixel purchase event in GTM"
**Ads Scripts**: "Create a Google Ads script to alert me when CPA exceeds $30"
