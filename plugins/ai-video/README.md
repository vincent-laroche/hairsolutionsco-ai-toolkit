# AI & Video Plugin

AI-powered video creation and delivery for Hair Solutions Co. — HeyGen avatar videos, content strategy, and Mux video infrastructure.

## Skills Included

| Skill | Purpose |
|-------|---------|
| `heygen` | AI avatar video generation — personalized videos at scale, interactive avatars |
| `video-cog` | Video content strategy — scripts, YouTube SEO, thumbnails, repurposing system |
| `mux-video` | Video hosting and delivery — uploads, HLS streaming, Shopify embed, analytics |

## Required Environment Variables

```bash
HEYGEN_API_KEY=your-heygen-api-key
MUX_TOKEN_ID=your-mux-token-id
MUX_TOKEN_SECRET=your-mux-token-secret
```
