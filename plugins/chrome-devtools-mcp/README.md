# Chrome DevTools MCP

Vendored plugin surface from
[ChromeDevTools/chrome-devtools-mcp](https://github.com/ChromeDevTools/chrome-devtools-mcp)
at version `1.2.0`, upstream commit
`4f8eb7ad6beecc58f56ec383f9ff43549a5604d4`.

It bundles the official Chrome DevTools MCP server configuration and six
upstream skills for browser debugging, accessibility, LCP optimization,
memory-leak analysis, troubleshooting, and CLI automation.
