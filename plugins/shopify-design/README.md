# Shopify & Design Plugin

Complete Shopify development and design toolkit for Hair Solutions Co. — from Liquid templating to UI systems and Figma workflows.

## Skills Included

| Skill | Purpose |
|-------|---------|
| `shopify-liquid` | Liquid syntax, objects, filters, tags, section schema |
| `shopify-theme-dev` | Theme architecture, sections, blocks, OS 2.0, Shopify CLI |
| `shopify-api` | Admin API, Storefront API, GraphQL, metafields, webhooks |
| `shopify-app-dev` | Remix framework, App Bridge, OAuth, extensions |
| `shopify-performance` | Core Web Vitals, image optimization, speed scoring |
| `shopify-debugging` | Liquid errors, JS debugging, theme customizer issues |
| `frontend-design` | Color systems, typography, spacing, CSS architecture |
| `ui-ux-pro-max` | 67+ component patterns, 96 palettes, conversion UX |
| `figma-automation` | Figma API, design token extraction, asset export |
