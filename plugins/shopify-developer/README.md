# Shopify Developer Plugin

Shopify theme development skills for `hairsolutions.co` on Horizon 3.5.1.

## Skills

| Skill | Use when |
|-------|----------|
| `shopify-liquid` | Writing, editing, or reviewing Liquid — sections, blocks, snippets, schema, dynamic sources, metaobjects |
| `shopify-front-end` | CSS, JavaScript, responsive layout, animations, Core Web Vitals, theme editor UI |
| `shopify-production` | Deployment, app safety, live theme operations, rollback, performance QA |

## Store

- **Store:** `one-head-hair.myshopify.com` / `hairsolutions.co`
- **Theme:** Horizon 3.5.1
- **Custom prefixes:** `hs-custom-*`, `support-hub-*`, `kb_*`, `blog-*`
- **Protected apps:** EComposer (`ecom-*`), SectionStore (`ss-*`), Foxify (`foxify-*`)
