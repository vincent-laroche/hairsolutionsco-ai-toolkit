# Reports & Dashboards — HubSpot How-To

## Create a Custom Report

1. **Reports → Reports → Create report**
2. Choose data source:
   - **Single object** — simple property-based report
   - **Funnel** — conversion between lifecycle stages
   - **Custom report builder** — cross-object, most powerful
3. Select properties/metrics, add filters (date range, pipeline)
4. Choose visualization: table / bar / line / pie
5. Save to a dashboard

## Create a Dashboard

1. **Reports → Dashboards → Create dashboard**
2. Name it, set access (private or team)
3. **Add report** → choose saved reports or create new
4. Arrange by dragging, set default date range
5. Save
