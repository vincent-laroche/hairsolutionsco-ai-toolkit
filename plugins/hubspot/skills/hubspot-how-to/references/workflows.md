# Workflows — HubSpot How-To

## Create a New Workflow

1. **Automation → Workflows → Create workflow**
2. Choose type: Contact-based (most common) / Deal-based / Ticket-based
3. Choose trigger: **When event occurs** (real-time) or **When filter criteria is met**
4. Set trigger conditions
5. Add actions with the **+** button
6. Set re-enrollment rules (prevents duplicate enrollment)
7. Test with a known contact
8. Toggle **On** to activate

## Key Actions

| Action | Use for |
|--------|---------|
| Send email | One-off triggered email |
| Delay | Wait X days/hours |
| Set property value | Update contact/deal property |
| Create task | Assign follow-up |
| Send internal email | Notify team |
| Enroll in sequence | Add to sales sequence |
| If/then branch | Split logic on conditions |

## Edit an Existing Workflow

1. Automation → Workflows → click workflow name
2. Click **Edit** (top right) — pauses the workflow
3. Make changes → **Review and publish** → confirm
4. Workflow auto-resumes

## Clone a Workflow

1. Find workflow → **Actions** (three dots) → **Clone**
2. Rename immediately, edit trigger/actions, activate

## Troubleshoot — Not Enrolling

- Check enrollment triggers — are criteria actually being met?
- Check re-enrollment settings — already enrolled?
- Check suppression lists
- Use **History** tab to see why specific contacts weren't enrolled
- Use **Test** button to simulate
