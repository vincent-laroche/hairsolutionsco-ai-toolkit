# Contacts & Lists — HubSpot How-To

## Import Contacts

1. **Contacts → Import** (top right) → **Import file** → upload CSV
2. Map columns to HubSpot properties
3. Set action: Create and update / Create only / Update only
4. Add to a static list (recommended for tracking the import)
5. Review and start

CSV requirements: Email column required. Include First Name, Last Name if available.

## Create an Active (Smart) List

1. **Contacts → Lists → Create list → Active list**
2. Name clearly (e.g. "Active Subscribers - Auto")
3. Set filter criteria → **Save list**
Active lists auto-update as contacts meet/leave criteria.

## Create a Static List

Same as above but choose **Static list**. Contacts stay until manually removed.

## Add a Custom Property

1. **Settings → Properties → Contact properties → Create property**
2. Set: Group, Label, Internal name
3. Choose field type: Single-line text / Dropdown / Checkbox / Date / Number
4. For dropdowns: add all options → Save

## Edit / Merge Contacts

- **Edit:** Open contact → click any property to edit inline
- **Merge:** Open contact → Actions → Merge → search duplicate → confirm
