# Marketing & Content Plugin

Full-stack marketing suite for Hair Solutions Co. — social media, paid ads, SEO, email sequences, content writing, and copywriting.

## Skills Included

| Skill | Purpose |
|-------|---------|
| `social-content` | Platform-optimized social media for LinkedIn, Instagram, TikTok, Twitter, Facebook |
| `paid-ads` | Google Ads, Meta, LinkedIn, TikTok ad campaigns — copy, targeting, bidding |
| `content-research-writer` | Blog posts, articles, product descriptions, email newsletters |
| `seo-audit` | Technical SEO audits, keyword research, on-page optimization, Core Web Vitals |
| `email-sequences` | Welcome series, abandoned cart, post-purchase, re-engagement flows |
| `copywriting` | Landing pages, sales copy, headlines, CTAs, product pages, brand voice |
